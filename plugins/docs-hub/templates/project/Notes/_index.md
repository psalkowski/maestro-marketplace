# Notes

WIP thoughts, meeting notes, scratchpad. Anything that doesn't fit Brainstorms / Specs / Plans.

## Frontmatter template (copy for a new note)

```yaml
---
type: note
status: draft     # optional — only if the note is worth tracking
date: YYYY-MM-DD  # optional
---
```

Free-form. Only `type: note` is required.
